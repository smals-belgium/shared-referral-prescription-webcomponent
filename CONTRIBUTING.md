## Who Can Contribute

Only recognized and authorized **health care provider IT solutions editors** (Integrators) may submit contributions.
If you are not an authorized Integrator, please contact Smals before submitting any contribution.

## Contribution Process

1. Fork the repository and create a feature branch.
2. Make your changes (code, documentation, or tests).
3. Ensure your contribution meets Smals’ quality criteria:

- Generic and reusable,
- Backward compatible (no regressions),
- Following existing coding standards.

4. Submit a Pull Request to the official repository: [https://github.com/smals-belgium/](https://github.com/smals-belgium/).
5. Smals, together with the sponsoring members, will review your contribution and may accept, reject, or request modifications.
